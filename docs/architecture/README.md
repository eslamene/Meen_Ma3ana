# Architecture Documentation

This directory contains core system architecture, technology stack, and business requirements documentation.

## Files

- **[TECH_STACK_OVERVIEW.md](./TECH_STACK_OVERVIEW.md)** - Comprehensive overview of the technology stack, architecture patterns, and important structural decisions. Essential reading for new developers.

- **[Meen-Ma3ana_BRD.md](./Meen-Ma3ana_BRD.md)** - Business Requirements Document (BRD) containing complete functional and non-functional requirements for the platform.

- **[APP_MAP_AND_MENU_STRUCTURE.md](./APP_MAP_AND_MENU_STRUCTURE.md)** - Application structure, navigation system, and menu organization documentation.

## Quick Start

New developers should start with:
1. [TECH_STACK_OVERVIEW.md](./TECH_STACK_OVERVIEW.md) - Understand the technology stack
2. [Meen-Ma3ana_BRD.md](./Meen-Ma3ana_BRD.md) - Learn business requirements
3. [APP_MAP_AND_MENU_STRUCTURE.md](./APP_MAP_AND_MENU_STRUCTURE.md) - Understand application structure
